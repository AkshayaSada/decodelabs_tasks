PROJECT NAME:
Image Text Recognition (OCR)

OBJECTIVE:
To recognize and extract text
from an image using OCR.

TOOLS USED:
Python
OpenCV
Pytesseract

WORKFLOW:
1. Load image
2. Convert to grayscale
3. Extract text
4. Display output

RESULT:
Successfully detected text
from sample image.

CONCLUSION:
This project demonstrates
basic OCR using pre-trained
recognition libraries.
